# Parallax Website

A desktop landing page showcasing smooth parallax scrolling effect using pure CSS.

### Features
- Pure CSS parallax scrolling effect
- Smooth scroll behavior
- Modern section-based layout

### Note
Designed for desktop view. Not responsive for mobile/tablet.

### Tech Stack
- HTML5
- CSS3
